RESTful Routes

A table of all 7 RESTful Routes

Name Path HTTP Verb Purpose
Index /aircrafts GET List all aircrafts
New /aircrafts/new GET Show new aircrafts form
Create /aircrafts POST Create a new aircrafts, then redirect to somewhere
Show /aircrafts/:id GET Show info about one specific aircraft
Edit /aircrafts/:id/edit GET Show edit form of the aircraft
Update /aircrafts/:id PUT Update a particular aircraft, then redirect to somewhere
Delete /aircrafts/:id DELETE Delete a particular aircraft, then redirect to somewhere
